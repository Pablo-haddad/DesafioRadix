Sobre o Projeto

Nosso projeto visa monitorar a extração de petróleo em tempo real, utilizando sensores modernos que garantem a precisão dos dados.

👨🏼‍💻

Como Funciona

Os sensores coletam dados e os transmitem para uma plataforma que permite análise detalhada, garantindo eficiência e segurança no processo de extração.
